# Studieo7 Family Salon & Bridal Studio - Trichy

A premium, fast, and SEO-optimized luxury website for India's First Digital Salon. This project is a complete 9-page vanilla HTML/CSS/JS build focused on luxury branding, local Trichy SEO, and a WhatsApp-first conversion funnel.

## 🌟 Key Features

*   **Luxury Aesthetic**: Dark theme with rich black backgrounds (`#0a0a0a`), champagne gold accents (`#c9a84c`), and frosted glassmorphism elements.
*   **WhatsApp-First Booking**: Floating WhatsApp buttons, mobile sticky bottom bars, and direct WhatsApp consultation pipelines strategically placed throughout the funnel.
*   **Bridal Focused**: A dedicated, dedicated high-converting bridal package page with tier comparisons, before/after sliders, and a scarcity-driven countdown timer.
*   **Performance & SEO**: Fully responsive mobile-first build, comprehensive structured data (LocalBusiness, ContactPage Schema), meta tags optimized for Trichy keywords, `sitemap.xml`, and `robots.txt`.
*   **Dynamic Interactions**: Fluid scroll-reveal animations, custom image lightbox, responsive masonry gallery, and an intelligent exit-intent popup offer.

## 🛠️ Technology Stack

*   **HTML5**: Semantic markup for excellent accessibility and SEO structure.
*   **CSS3**: Vanilla CSS utilizing CSS Variables (`:root`) acting as a lightweight design system. No external frameworks used for maximum performance.
*   **JavaScript (ES6)**: Vanilla JS handling complex UI states (Accordion, Modals, Lightbox, Scroll Tracking, Form Validation) without reliance on heavy libraries like jQuery.

## 🚀 Running the Project

Because this is a vanilla HTML/CSS/JS application, no build steps are required. 

1. Simply open `index.html` in any modern web browser to view the site locally.
2. For the best development experience, consider using a local server (like the VS Code Live Server extension) to avoid any CORS issues related to fetching local assets if you expand the project later.

## 🎨 Design System Overview

*   **Primary Fonts**: `Playfair Display` (Headings, Serif) and `Inter` (Body, Sans-Serif).
*   **Core Colors**: 
    *   Primary Background: `#0a0a0a`
    *   Gold Accent: `#c9a84c`
    *   Text Muted: `#b0b0b0`
    *   Glass Panel: `rgba(255, 255, 255, 0.05)`
*   **Key Components**: Glass Cards (`.glass-card`), Shimmer Buttons (`.btn-shimmer`), Sticky Scrolled Header (`.header.scrolled`), and Staggered Animations (`.stagger-children`).
